# User Stories

## Epic: Governed SDLC Execution

US-01
As a project owner,
I want to submit a project description and see the current SDLC stage,
so that I can verify the system captured my intent.

Acceptance Link: AC-01

US-02
As a project owner,
I want to request approval for the requirements stage,
so that no implementation starts before I confirm scope.

Acceptance Link: AC-02

US-03
As the orchestrator,
I want to start and pause runs,
so that autonomous execution remains controllable.

Acceptance Link: AC-03

US-04
As the orchestrator,
I want each run transition logged with timestamps and context,
so that I can audit what happened after the fact.

Acceptance Link: AC-04

US-05
As a reviewer,
I want to see the generated PRD and user stories before approval,
so that I can validate requirements coverage.

Acceptance Link: AC-05
