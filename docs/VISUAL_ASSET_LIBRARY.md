# Visual Asset Library

## Purpose

This document indexes the reusable SVG-style visuals for the **Agentic SDLC Platform**.

These assets are designed for:

- website hero sections
- product landing pages
- investor decks
- startup pitch decks
- thesis presentations
- research and architecture appendices

The goal is to give the project a consistent visual language across academic, technical, and commercial material.

## Visual Direction

The SVG set uses one shared visual system:

- deep navy foundations
- cool blue and teal gradients
- coral highlights for risk and review
- soft glass-like cards
- control-plane and dashboard motifs

This makes the visuals feel more like a serious software platform than a generic AI illustration.

## Practical Note

These visuals are intentionally presentation-ready rather than screenshot-dependent.

That means:

- they are clean enough for slides
- they can be recolored later
- they can be simplified further for paper figures
- they can be imported into Figma, PowerPoint, or draw.io for refinement
