# Portfolio App Pack

## Goal
Build a responsive personal portfolio with clean navigation, project showcase, and contact form.

## Functional Requirements
- FR-001: Home section with hero headline and CTA.
- FR-002: About section with bio and skills.
- FR-003: Projects section with at least 4 cards.
- FR-003A: Projects section must include one explicit "Data Engineering Pipeline" project card (title, short description, tech tags, and CTA link/button).
- FR-004: Contact section with name/email/message form.
- FR-005: Top navigation for desktop and hamburger menu for mobile.
- FR-006: Active nav link highlighting based on section in view.

## Quality Requirements
- QR-001: Responsive from 320px to 1440px.
- QR-002: Keyboard accessible nav and form controls.
- QR-003: Lighthouse accessibility >= 90.
- QR-004: Page load <= 2.5s on local preview.

## Task Seeds
- Implement responsive nav shell and hamburger menu.
- Build Home/About/Projects/Contact sections.
- Add "Data Engineering Pipeline" card in Projects with ETL/warehouse/observability context.
- Add active-link and smooth scroll behavior.
- Add form validation and success/error states.
- Add scoped tests for `index.html` and key interactions.
