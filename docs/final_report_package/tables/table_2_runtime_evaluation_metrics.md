| Metric | Description | Purpose |
|---|---|---|
| Patch success rate | Fraction of code mutation attempts resulting in accepted patch artifacts | Measures implementation reliability |
| Recovery success rate | Fraction of recovery attempts that restore forward progress | Measures autonomous repair effectiveness |
| Time-to-green | Elapsed time from run start to passing validation outcome | Measures delivery velocity |
| Validation pass rate | Ratio of runs/work-items that pass validation gates | Measures quality guard conformance |
| Average retries | Mean retry count per run/work-item | Measures instability and repair burden |
| Context efficiency ratio | Selected context items / loaded context items | Measures context selection quality and token discipline |
| Safe-change success rate | Fraction of scoped/bounded changes merged without policy violation | Measures governance-safe autonomy |
| Budget utilization | Consumed AI/runtime budget vs configured limits | Measures bounded execution control |
