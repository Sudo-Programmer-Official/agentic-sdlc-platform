# College Website Pack

## Goal
Build an informational college website with academics, admissions, events, and contact flow.

## Functional Requirements
- FR-001: Landing page with college overview.
- FR-002: Academics page/section with departments and programs.
- FR-003: Admissions page/section with process steps.
- FR-004: Events/news section with cards and dates.
- FR-005: Contact and inquiry form.
- FR-006: Global navigation + mobile drawer.

## Quality Requirements
- QR-001: Information architecture is clear and scan-friendly.
- QR-002: WCAG-friendly focus states and semantic headings.
- QR-003: Responsive grids for cards and lists.
- QR-004: No broken links in navigation flow.

## Task Seeds
- Build section structure and route/anchor flow.
- Implement admissions timeline component.
- Implement events listing with mock data.
- Add form validation and tests.
