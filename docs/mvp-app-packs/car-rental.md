# Car Rental Website Pack

## Goal
Build a car rental app with vehicle listings, filter/search, and booking form flow.

## Functional Requirements
- FR-001: Vehicle listing grid with card details.
- FR-002: Filter by category, price range, and availability.
- FR-003: Vehicle detail view/section.
- FR-004: Booking form with pickup/drop-off dates.
- FR-005: Pricing summary and booking confirmation state.
- FR-006: Responsive navigation with mobile menu.

## Quality Requirements
- QR-001: Fast filtering (<200ms local interaction target).
- QR-002: Form validation for date and required fields.
- QR-003: Mobile usability for filters and booking CTA.
- QR-004: Deterministic test coverage for key booking flow.

## Task Seeds
- Build listing and filters.
- Implement booking form and summary.
- Add state management for selected vehicle.
- Add tests for filter + booking validations.
