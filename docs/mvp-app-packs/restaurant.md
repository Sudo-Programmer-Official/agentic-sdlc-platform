# Restaurant Website Pack

## Goal
Build a responsive restaurant website with menu, reservations CTA, and location/contact details.

## Functional Requirements
- FR-001: Hero section with restaurant branding.
- FR-002: Menu section grouped by category.
- FR-003: Featured dishes carousel/grid.
- FR-004: Reservation CTA linking to booking action.
- FR-005: Hours, address, map link, and contact details.
- FR-006: Mobile-first nav with hamburger behavior.

## Quality Requirements
- QR-001: Responsive layout and readable typography.
- QR-002: Accessible color contrast and keyboard nav.
- QR-003: Performance budget for images.
- QR-004: Stable render across Chrome and Safari.

## Task Seeds
- Build content sections and reusable cards.
- Add nav + mobile menu behavior.
- Optimize images and lazy loading.
- Add smoke tests for section visibility and links.
